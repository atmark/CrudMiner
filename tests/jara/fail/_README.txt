Installation Guide and Readme

Stage 1. Server requirements
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Jara v1.6 requires all the following on the server:

	* A mechanism to upload files to the server (e.g. FTP)
	* PHP 5.2.x/5.3.x with mysqli installed and enabled
	* MySQL 5.x
